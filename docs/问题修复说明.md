# Tkinter错误修复说明

## 问题描述

运行代码时出现以下错误：

```
RuntimeError: main thread is not in main loop
Exception ignored in: <function Variable.__del__ at 0x000001DBB6A05EE0>
Traceback (most recent call last):
  File "C:\D\code\python\Pythons\py12\Lib\tkinter\__init__.py", line 414, in __del__
RuntimeError: main thread is not in main loop
```

## 原因分析

1. **Matplotlib默认后端**: Tkinter (TkAgg)
2. **问题场景**:
   - 在非主线程中运行代码
   - 在Jupyter Notebook中运行
   - 在脚本/批处理环境中运行
3. **Tkinter限制**: 必须在主线程中使用

## 解决方案

使用**非交互式后端 (Agg)** 替代Tkinter后端。

### 修改的文件

#### 1. reshaping_analyzer.py
```python
import matplotlib
matplotlib.use('Agg')  # 在import pyplot之前设置
import matplotlib.pyplot as plt
```

#### 2. main_analysis.py
```python
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt
```

#### 3. usage_example.py
```python
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt
```

## Agg后端的特点

### ✅ 优点
- 不依赖GUI（Tkinter/Qt）
- 可在任何环境运行（脚本/Jupyter/服务器）
- 不会弹出窗口
- 适合批量生成图表

### ⚠️ 注意事项
- **不能显示图表**：必须用`plt.savefig()`保存
- **不能交互**：不支持缩放、平移等交互功能

## 使用建议

### 场景1: 生成报告图表（推荐Agg）
```python
matplotlib.use('Agg')
# 生成图表
plt.savefig('chart.png')
plt.close()  # 关闭图表释放内存
```

### 场景2: 交互式分析（使用默认）
```python
# 不设置后端，使用默认的Tkinter
plt.plot(x, y)
plt.show()  # 显示交互式窗口
```

### 场景3: Jupyter Notebook
```python
%matplotlib inline
# 或
%matplotlib widget
```

## 常见问题

### Q1: 图表中文显示为方框怎么办？
A: 确保设置了中文字体：
```python
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
```

### Q2: 如何切换回交互模式？
A: 注释掉`matplotlib.use('Agg')`即可

### Q3: 批量生成很多图表时内存占用高？
A: 每次保存后调用`plt.close()`：
```python
plt.savefig('chart.png')
plt.close()  # 释放内存
```
