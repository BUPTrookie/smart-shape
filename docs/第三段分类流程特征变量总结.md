# 第三段分类流程特征变量总结

## 概述
本文档总结了第三段分类流程中涉及的4个核心特征变量及其在分类决策中的应用。

## 特征变量详细说明

### 1. 波峰波谷特征
```python
len(peaks)     # 峰值数量
len(valleys)   # 谷值数量
```

- **特征来源**：`find_peaks()` 函数检测结果
- **用途**：判断是否存在明显的峰谷波动
- **判断条件**：`len(peaks) >= 1` 且 `len(valleys) >= 1`
- **分类目标**：WAVE（波形）

### 2. 振幅特征
```python
amplitude      # 振幅 = max(p_values) - min(p_values)
```

- **特征来源**：数据段的极值差
- **双重用途**：
  - **波形判断**：`amplitude >= 0.08`（波形需要有足够振幅）
  - **平坦判断**：`amplitude <= 0.065`（振幅小认为是平坦）
- **计算方式**：数据段的最大值减最小值

### 3. 凸凹判断特征
```python
upOrDown       # 凸凹判断函数返回值
```

- **特征来源**：`calculateUpOrDown()` 复合函数
- **返回值含义**：
  - `1`：确认为上凸（ARC_UP）
  - `2`：确认为下凹（ARC_DOWN）
  - `0`：不明确（不满足凸凹条件）
- **判断逻辑**：基于全局均值的距离比较 + 左右侧趋势比例

### 4. 曲率特征
```python
a              # 二次拟合的二次项系数
```

- **特征来源**：`np.polyfit(x, y, 2)[0]`
- **用途**：作为备选的凸凹判断条件
- **判断条件**：
  - `< -0.0008`：负曲率 → 上凸（ARC_UP）
  - `> 0.0008`：正曲率 → 下凹（ARC_DOWN）
- **物理意义**：描述曲线的弯曲方向和程度

## 特征在分类决策中的应用优先级

### 决策树结构
```
1. WAVE判断（最高优先级）
   ├─ len(peaks) >= 1
   ├─ len(valleys) >= 1
   └─ amplitude >= 0.08

2. ARC_UP判断（次高优先级）上凹
   ├─ upOrDown == 1
   └─ a < -0.0008（备选）

3. ARC_DOWN判断（次高优先级）#下凹
   ├─ upOrDown == 2
   └─ a > 0.0008（备选）

4. FLAT判断（第三优先级）
   └─ amplitude <= 0.065

5. FLAT兜底（最低优先级）
   └─ 其他所有情况
```

## 特征类型分类

### 直接判断特征
- **峰谷数量**：定性判断是否有波动
- **振幅**：定量判断波动强度
- **曲率**：数学描述弯曲方向

### 复合判断特征
- **upOrDown函数**：综合了极值位置、变化趋势、比例阈值的复合特征

## 计算依赖关系

```
p_values (P9-P16)
├── amplitude = max(p_values) - min(p_values)
├── a = polyfit(x, y, 2)[0] (二次拟合)
```

## 特征阈值总结表

| 特征变量 | 分类目标 | 阈值 | 用途 |
|---------|---------|------|-----|
| len(peaks) | WAVE | >= 1 | 峰值存在 |
| len(valleys) | WAVE | >= 1 | 谷值存在 |
| amplitude | WAVE | >= 0.08 | 波形振幅 |
| amplitude | FLAT | <= 0.065 | 平坦振幅 |
| upOrDown | ARC_UP | == 1 | 上凸判断 |
| upOrDown | ARC_DOWN | == 2 | 下凹判断 |
| a | ARC_UP | < -0.0008 | 负曲率 |
| a | ARC_DOWN | > 0.0008 | 正曲率 |

## 分类逻辑特点

### 1. 多重保障机制
- 主要条件失效时，有备选条件兜底
- 例如：upOrDown 失效时，使用曲率 a 作为备选

### 2. 优先级清晰
- WAVE 优先于 ARC 和 FLAT
- 复合判断（upOrDown）优先于单一判断（曲率）
- 明确条件优先于兜底条件

### 3. 阈值设定逻辑
- 波形需要更严格的条件（峰谷+振幅）
- 平坦条件相对宽松（仅振幅）
- 凸凹判断有主要和备选两套方案
