整形压头影响量化分析系统运行流程
一、系统概述
功能: 从历史数据学习整形压头对测量点位的影响规律,建立预测模型
输入: 历史加工数据(Pre/Post配对数据)
输出: 压头影响系数、预测模型、性能评估报告
二、详细运行流程(8个步骤)
步骤1: 数据加载与Pre/Post配对
目标: 从原始数据中提取同一工件的整形前后测量数据 执行过程:
读取 Data/total_final_processed.xlsx
按 FAI156(条码)分组,匹配Pre/Post状态
计算20个点位的Delta变化量: Δ = Post - Pre
提取压头参数(RS1-RS4的X/Z坐标)
数据清洗:填充缺失压头值,检测极端值
输出:
配对样本: 2567对
原始记录: 5304条
步骤2: 特征工程
目标: 构造用于机器学习的特征矩阵 构造26个特征:
(1) 位置特征 (13个)
为每个压头在不同位置构造特征
例如: RS1_pos_-32.25 = RS1Z × (RS1X == -32.25)
含义: 压头在该位置的使用强度
(2) Pre曲线特征 (7个)
pre_mean: 全局均值
pre_std: 全局标准差
pre_slope: 整体斜率
pre_seg1/2/3/4_mean: 各段均值
(3) 交互特征 (6个)
统计高频压头组合
例如: inter_RS1_RS4 = RS1Z × RS4Z
含义: 压头同时使用的交互效应
输出:
特征矩阵 X: (2567, 26)
目标变量 y: (2567, 20) - delta_P1到delta_P20
步骤3: 划分训练集/测试集
目标: 独立划分数据,避免数据泄露 执行过程:
随机划分: 80%训练集(2053条), 20%测试集(514条)
使用固定随机种子(random_state=42)确保可复现
数据分布诊断:
对比训练集/测试集的压头使用率
对比Delta值的标准差
输出:
X_train, X_test
y_train, y_test
步骤4: 模型训练与交叉验证
目标: 选择最佳超参数并训练模型 执行过程:
特征标准化: 使用StandardScaler(仅在训练集上fit)
5折交叉验证: 测试不同Alpha值
Alpha候选: [0.001, 0.01, 0.1, 1.0, 10.0, 100.0]
评估指标: R²(决定系数)
选择最佳Alpha: 1.0
训练最终模型: MultiOutputRegressor + Ridge回归
输出:
训练好的模型
最佳Alpha = 1.0
步骤5: 测试集评估
目标: 评估模型泛化能力 执行过程:
使用训练集的scaler标准化测试集
模型预测测试集
计算评估指标:
R²(决定系数): 拟合优度
RMSE(均方根误差): 预测精度
输出:
测试集性能指标
20个点位各自的R²和RMSE
步骤6: 提取影响系数
目标: 量化每个压头在每个位置对点位的影响 执行过程:
从训练好的Ridge模型中提取系数
按压头和位置组织数据
计算截距(基准值)
输出:
influence_coefficients.json
结构: influence[压头名][位置] = {P1: 系数, ..., P20: 系数}
步骤7: 导出结果
目标: 保存所有分析结果 生成的文件:
(1) 影响系数
文件: influence_coefficients.json
内容: 压头对每个点位的影响权重
(2) 评估指标
文件: model_metrics_train.json / model_metrics_test.json
内容: 每个点位的R²和RMSE
(3) 预测结果
文件: predictions_train.csv / predictions_test.csv
内容: 真实值、预测值、误差
(4) 特征数据
文件: training_data.csv
内容: 完整特征矩阵(2567行 × 27列)
步骤8: 生成分析报告
目标: 汇总关键发现 报告内容:
(1) 数据统计
原始记录数、配对样本数
训练集/测试集分布
(2) 模型性能
训练集: 平均R² = 0.7744, RMSE = 0.0192
测试集: 平均R² = 0.8819, RMSE = 0.0149
点位R² Top5: P10(0.9741), P11(0.9745), P12(0.9723)
(3) 过拟合检测
R²差异 = -0.1075 (测试集优于训练集)
异常诊断: 训练集包含更多难预测样本
(4) Top5最大偏差样本
识别预测最失败的5个工件
显示条码、总误差、最大误差点位
(5) 影响系数概览
各压头的位置档位数量
三、关键技术点
1. 数据配对策略
Pre/Post匹配: 确保同一工件的前后对比
Delta计算: Δ = Post - Pre (整形前后变化量)
2. 特征工程
位置特征: 捕捉压头在不同位置的使用强度
Pre曲线特征: 反映工件初始状态
交互特征: 捕捉压头组合效应
3. 模型选择
MultiOutputRegressor: 同时预测20个点位
Ridge回归: L2正则化,防止过拟合
交叉验证: 选择最佳正则化参数
4. 数据泄露防护
独立划分: 测试集完全独立
标准化隔离: Scaler仅在训练集上fit
四、输出文件结构

Output/RS_impact_analysis/

├── predictions_train.csv         # 训练集预测结果
├── predictions_test.csv          # 测试集预测结果
└── training_data.csv             # 完整特征数据
